# simple-frontend

## The application has already been built for you

This is the only time you should need git

```
git clone https://github.com/cts-workshop-02-2019/angular7-m0
cd angular7-m0
```

## Deploy the app

Verify I'm pointed to the right space:
```
cf target
```

Push the app:
```
cf push
```
